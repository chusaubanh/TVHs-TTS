===================================================
           Thành Vinh Studio v2.0
===================================================

This is a standalone application for High-Quality Vietnamese Text-to-Speech.

[ HOW TO RUN ]
---------------------------------------------------
Simply double-click:
   -> start.bat

This will launch both the AI Backend and the Web Interface.


[ INSTALLATION ]
---------------------------------------------------
If this is the first time running on a new computer:
1. Double-click: install.bat
   (This sets up Python and Node.js environments)

2. Then run start.bat


[ CUSTOM MODELS ]
---------------------------------------------------
To add more voices:
1. Download models/checkpoints (e.g., from HuggingFace).
2. Place them in the 'models/' folder (or configure path in backend).
3. Restart the app.
