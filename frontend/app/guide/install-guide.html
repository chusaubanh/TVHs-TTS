2. Cài đặt prerequisites

2.1. Python 3.10+

Tải installer tại https://www.python.org/downloads/

Quan trọng

Khi cài, phải tick "Add Python to PATH" ở bước đầu tiên.

Kiểm tra sau khi cài:

CMD / PowerShell python --version
# Mong đợi: Python 3.10.x trở lên
Terminal # Cài Homebrew (nếu chưa có)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Cài Python
brew install python@3.12
Terminal # Ubuntu / Debian
sudo apt update && sudo apt install -y python3.12 python3.12-venv python3-pip

# Fedora
sudo dnf install -y python3.12

2.2. Node.js 18+

Tải LTS installer tại https://nodejs.org/ (v20.x)

CMD / PowerShell node --version
# Mong đợi: v20.x.x trở lên
Terminal brew install node@20
Terminal # Ubuntu / Debian
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

2.3. eSpeak NG Bắt buộc

Thư viện phonemization — chuyển văn bản tiếng Việt thành phiên âm. Ứng dụng sẽ không hoạt động nếu thiếu.

Tải MSI installer tại https://github.com/espeak-ng/espeak-ng/releases

Chạy installer, để mặc định đường dẫn C:\Program Files\eSpeak NG\

Tự động

install.bat sẽ tự động tải và cài eSpeak NG nếu chưa có.

Terminal brew install espeak-ng
Terminal # Ubuntu / Debian
sudo apt install -y espeak-ng

# Fedora
sudo dnf install -y espeak-ng

2.4. uv (Astral) Script tự cài

Trình quản lý Python package nhanh hơn pip. Script install.bat / install.sh sẽ tự động cài nếu chưa có.

PowerShell powershell -ExecutionPolicy Bypass -c "irm https://astral.sh/uv/install.ps1 | iex"
Terminal curl -LsSf https://astral.sh/uv/install.sh | sh

3. Cài đặt dự án

Cách 1: Script tự động Khuyến nghị

Script sẽ tự động kiểm tra prerequisites, cài Python dependencies, Node dependencies, eSpeak NG và hỏi trước khi tải OmniVoice.

CMD :: Clone dự án
git clone https://github.com/chusaubanh/ThanhVinhStudio
cd ThanhVinhStudio

:: Cài đặt dependencies, eSpeak NG và model tùy chọn
install.bat

:: Khởi động
start.bat
Terminal # Clone dự án
git clone https://github.com/chusaubanh/ThanhVinhStudio
cd ThanhVinhStudio

# Cấp quyền chạy script
chmod +x install.sh start.sh

# Cài đặt tất cả
./install.sh

# Khởi động
./start.sh

Cách 2: Cài thủ công

Nếu script tự động gặp lỗi, bạn có thể cài từng bước:

1

Clone dự án

git clone https://github.com/chusaubanh/ThanhVinhStudio
cd ThanhVinhStudio
2

Cài Python dependencies (Backend)

uv venv # Tạo virtual environment
uv sync  # Cài tất cả Python packages
3

Cài Node.js dependencies (Frontend)

cd frontend
npm install
cd ..
4

Tải model OmniVoice Tùy chọn

Model OmniVoice (~1.5GB) cần cho tab OmniVoice, voice design, voice clone và TTS đa ngôn ngữ. Nếu chưa tải, VieNeu-TTS vẫn chạy bình thường.

uv run python -c "from huggingface_hub import snapshot_download; snapshot_download('k2-fsa/OmniVoice', local_dir='models/omnivoice', ignore_patterns=['*.md','*.txt'])"
5

Khởi động ứng dụng

# Terminal 1 — Backend
uv run python -m backend.main

# Terminal 2 — Frontend
cd frontend && npm run dev

Mở trình duyệt: http://localhost:3000

4. Khởi động ứng dụng

Chế độ phát triển (dev)

CMD start.bat

Script sẽ:

  • Kill process cũ trên port 8000 (nếu có)
  • Khởi động backend tại http://localhost:8000
  • Đợi backend sẵn sàng (tối đa 30s)
  • Khởi động frontend tại http://localhost:3000
  • Tự động mở trình duyệt
Terminal ./start.sh

Chế độ production (static)

Build frontend thành static files, backend FastAPI sẽ serve trực tiếp:

Terminal cd frontend
npm run build  # Tạo frontend/out/
cd ..
uv run python -m backend.main  # Serve tại http://localhost:8000

Mở http://localhost:8000 — không cần chạy frontend dev server.

5. Tải model

VieNeu-TTS (Base model)

Model chính cho tổng hợp giọng nói tiếng Việt. Có thể tải qua UI hoặc command line:

Terminal # Tải qua API
curl -X POST http://localhost:8000/v1/download/base

# Kiểm tra tiến trình
curl http://localhost:8000/v1/download/progress

OmniVoice Tùy chọn

Engine TTS 0.6B, hỗ trợ 600+ ngôn ngữ. Trong UI hiện đã ưu tiên các lựa chọn thường dùng của team: Vietnamese, English, Chinese, Japanese, Korean, French, German, Khmer, Indonesian, Thai, Malay, Singapore, Mexico và Bangladesh.

Terminal curl -X POST http://localhost:8000/v1/download/omnivoice
Vietnamese Khmer Indonesian Thai Malay Singapore / English Mexico / Spanish Bangladesh / Bengali

LoRA Adapter Tùy chọn

Adapter tinh chỉnh giọng nói cụ thể. Tải qua UI (sidebar Studio) hoặc command line:

Terminal curl -X POST http://localhost:8000/v1/download/lora -H "Content-Type: application/json" -d '{"name": "ngoc-huyen"}'
Lưu ý

Sau khi tải xong, nhấn nút "Tải lại model" trong UI hoặc gọi POST /v1/models/reload để nạp model vào bộ nhớ.

6. Workflow sau khi cài

1

Mở Studio và chọn engine

Dùng GGUF cho máy CPU, PyTorch khi cần GPU/LoRA, hoặc Turbo khi ưu tiên tốc độ.

2

Tạo giọng tiếng Việt bằng VieNeu

Nhập kịch bản, chọn preset voice, thêm pause marker nếu cần rồi nghe lại bằng player trước khi tải file WAV.

3

Dùng OmniVoice khi cần đa ngôn ngữ

Load OmniVoice trong tab riêng, chọn ngôn ngữ, dùng voice design hoặc upload audio tham chiếu để clone giọng.

Endpoint kiểm tra nhanh

Sau khi backend chạy, mở http://localhost:8000/health để xem model, CUDA, eSpeak, thư mục output và trạng thái OmniVoice.

7. Cấu hình GPU (tùy chọn)

NVIDIA GPU (CUDA)

Để sử dụng engine PyTorch (GPU) cho chất lượng cao nhất và hỗ trợ LoRA:

  1. Cài NVIDIA Driver phiên bản 570.65+ (CUDA 12.8)
  2. CUDA toolkit sẽ được cài tự động qua PyTorch
  3. Trong UI, chọn engine PyTorch (GPU) ở sidebar Studio
Kiểm tra CUDA uv run python -c "import torch; print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0) if torch.cuda.is_available() else 'No GPU')"

Apple Silicon (Metal)

Trên macOS với chip M1/M2/M3, PyTorch sử dụng Metal Performance Shaders (MPS) tự động. Hiện tại OmniVoice hỗ trợ CPU trên macOS.

8. Build standalone (tùy chọn)

Đóng gói ứng dụng thành file .exe (Windows) hoặc ứng dụng macOS:

Terminal # Build frontend trước
cd frontend && npm run build && cd ..

# Build standalone app
uv run python build.py

Output: dist/ThanhVinhStudio/ — thư mục chứa ứng dụng standalone.

Yêu cầu build

Cần cài thêm: pip install pywebview pyinstaller (đã có trong requirements-build.txt)

9. Khắc phục lỗi thường gặp

Lỗi: "vieneu package not found"

Nguyên nhân

Chưa cài đặt vieneu hoặc chưa kích hoạt virtual environment.

Giải pháp uv sync  # Cài lại dependencies

Lỗi: "cannot import name 'Normalizer' from 'sea_g2p'"

Nguyên nhân

Package sea-g2p bị cài thiếu wrapper Python, thường xảy ra khi môi trường ảo bị restore hoặc install bị ngắt giữa chừng.

Giải pháp uv pip install --force-reinstall sea-g2p==0.7.5
uv run python -c "from sea_g2p import Normalizer; print(Normalizer)"

Lỗi: "eSpeak NG not found"

Nguyên nhân

Chưa cài eSpeak NG hoặc chưa thêm vào PATH.

Windows — kiểm tra PATH where espeak-ng
# Nếu không thấy, thêm C:\Program Files\eSpeak NG\ vào PATH

Lỗi: Port 8000 đã được sử dụng

Windows — tìm và kill process netstat -aon | findstr :8000
taskkill /F /PID <PID>

macOS / Linux lsof -ti:8000 | xargs kill

Lỗi: "Cross origin request detected"

Đây là warning, không phải lỗi

Next.js cảnh báo khi frontend ở localhost:3000 gọi API ở localhost:8000. Đã được fix trong next.config.ts với allowedDevOrigins.

Lỗi: "Method Not Allowed" khi gọi API

Nguyên nhân

Backend cũ chưa restart sau khi cập nhật code. Cần restart backend.

Lỗi: npm install chậm hoặc timeout

Giải pháp # Dùng registry mirror (cho người dùng Việt Nam)
npm config set registry https://registry.npmmirror.com
npm install

Kiểm tra trạng thái hệ thống

API # Health check chi tiết (model, CUDA, eSpeak, output folder)
curl http://localhost:8000/health

# Trạng thái model
curl http://localhost:8000/v1/status
