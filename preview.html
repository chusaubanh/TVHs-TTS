Tổng quan

Thành Vinh Studio hỗ trợ ba luồng chính: chọn preset voice, clone từ audio mẫu và dựng dialogue nhiều dòng.

Preset voices

Dùng khi bạn muốn tạo audio nhanh từ model sẵn có. Chỉ cần nhập text, chọn model voice và tinh chỉnh.

Chọn voice preset → nhập text → bấm Tạo giọng → nghe bằng player.

Voice Clone

Upload file audio mẫu 3-10 giây, nhập transcript đúng với file.

Nếu ref_text sai, kết quả clone sẽ lệch rõ rệt.
